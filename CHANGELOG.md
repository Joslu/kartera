## [1.0.0] - 2026-01-29
### Added
- Payment methods as entities (cash/debit/credit) with admin UI.
- Credit cards with cutoff, days-after-cutoff payment window, and cycle summaries.
- Credit card cycle detail view (spent vs payments).
- Debit/Cash balances (income minus expenses).
- Income destination account (paymentMethodId) and UI selection.
- Transaction filter by payment method.
- Cards screen (credit + debit) and payment window indicator.
- About page styled as terminal with ASCII.
